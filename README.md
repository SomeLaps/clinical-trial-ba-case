# Анализ процесса клинических исследований (фаза III)

Кейс-стади по системному анализу регулируемого бизнес-процесса с расчётом эффекта от автоматизации.

## Суть проекта
Выявлены 3 узких места этапа проведения клинических исследований лекарственных препаратов:
- Последовательное согласование протокола (критический путь 74+ дня)
- Ручной ввод данных (12–18 мин/визит, 4–14 ошибок на 10 000 полей)
- Отсутствие прогнозирования остатков препарата (12% простоев)

## Результаты
- Предложены 3 решения с обоснованием через нормативную базу (ICH GCP, 61-ФЗ)
- Рассчитан финансовый эффект: **877 500 ₽ за исследование**
- Срок окупаемости решений: от <1 месяца до 3 месяцев

## Артефакты
- Диаграмма BPMN 2.0 (полную цепочку этапов с ролями)
- Диаграмма IDEF0 (контекстная модель)
- Диаграмма EPC (логика решений и контрольные точки)

*Проект выполнен в учебных целях для отработки навыков бизнес-анализа в регулируемых отраслях.*
